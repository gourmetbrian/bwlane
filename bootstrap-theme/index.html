#NeverTrump Java Game

I created a two-person, turn-based strategy game in Java. Players connect to one another over a server, and play each side on their own client. Player actions are converted from Java to XML code, passed to the server, then on to the other client, where they are translated back into Java. Developed using Gourmet Gaming Java codebase. As this code is proprietary, the GitHub repo is private. Based on Old Maid, object of the game is to not end up with Trump.
